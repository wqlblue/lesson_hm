# 模板编译

- 正则
  - 开发中的一项单独技能，如写sql
  - 在字符串中，长文中，匹配想要的内容
  - 编写正则表达式
    - 一个字符串 1 
    - [0-9] 0-9 一个数字字符
    - {10} 长度10  一个字符的匹配次数
    - ^ 开头匹配
    - $ 结尾匹配
    - + 一次或多次
    - * 0或多次
    - ? 0或1次
