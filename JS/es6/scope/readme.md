##  变量
- 作用域
    变量不会单独存在，它属于一个作用域
    作用域是变量的查找规则  在当前作用域中查找变量，找不到，就去父级作用域中查找，直到找到为止

##  var a = 1; 哪些东西为它工作

 - 编译器   CTO
     var  a  =  1  分词
 - 作用域   COO 运营经理
     变量属于作用域
     作用域链
